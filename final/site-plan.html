Dynamic elements

I want to use js for a list of my projects where all I have to do is have an id or class in html and then the js will fill up the rest with the information I give it. I would also want the projects to be displayed dynamically but I'm not sure how I would want to do that yet. And since I am fluent in spanish I want there to be a button to change the Page and everything in it to be in spanish
